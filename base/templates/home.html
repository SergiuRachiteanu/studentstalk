Students Talk




A real-time chat application for students.



Welcome to Students Talk, an open source project designed for the students of West University of Timisoara.

Are you tired of not getting a fast response when you need it the most? Well, here you can get rid of this problem, due to the friendly mechanism of this app. While in other apps you need to add friends to conversate or to be in a group, here you have preset channels and all you have to do is connect to one of these, and you can already start chatting.

To be more precise, each channel represents one of the 11 faculties from the West University of Timisoara. Furthermore, each channel has it's own subchannels, for certain purposes
(e.g: socialising, exams, homeworks,etc.), but I let you to discover that.

This app is django-based with a MySQL database connection. This is version 1.0, more features will be added soon.

Enjoy chatting!
